## Setup

Add the `jcenter` repository in your root `build.gradle`.
```groovy
allprojects {
    repositories {
        jcenter()
    }
}
```

Add the dependencies in your app level `build.gradle`.
```groovy
dependencies {
    implementation 'com.codezjx.library:andlinker:0.6.0'
}
```
